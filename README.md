# Portfolio 2026 - Mehdi Belmiloud

Bienvenue sur le dépôt de mon portfolio personnel ! Ce site interactif et moderne a été conçu pour présenter mon profil, mon parcours et mes projets en tant qu'étudiant en **BTS SIO SLAM**, avec pour objectif de décrocher une **alternance pour l'année 2026**.

**[Voir le Portfolio en ligne](https://portfolio-mehdi-belmiloud.vercel.app/)**

---

## Fonctionnalités Principales

- **Thème Sombre / Clair** : Détection automatique des préférences du système et bascule manuelle avec sauvegarde dans le `localStorage`.
- **Entièrement Responsive** : Interface fluide s'adaptant parfaitement à tous les écrans (Mobile, Tablette, Desktop).
- **Design Moderne & Glassmorphism** : Utilisation de cartes avec effets de verre poli, gradients floutés et éléments géométriques interactifs en arrière-plan.
- **Animations Interactives (Vanilla JS & CSS)** :
  - Effet "Machine à écrire" (Typewriter) dynamique sur la page d'accueil et le footer.
  - Apparition fluide des éléments au défilement (*Scroll Reveal*).
  - Effets visuels immersifs qui suivent le curseur de la souris ("Spotlight" global et "Aurora" au survol des cartes).
  - Animation personnalisée "Easter Egg" (référence Gojo) sur la page de contact avec effets de glitch et de brillance.
- ** Formulaire de Contact Actif** : Intégration de *Formspree* permettant de recevoir des messages directement par email sans nécessiter de backend propre.

## Technologies Utilisées

- **Frontend** : HTML5, Vanilla JavaScript (ES6+)
- **Styling** : Tailwind CSS (via CDN) & CSS3 (variables personnalisées, animations avancées, custom scrollbars)
- **Icônes** : Font Awesome 6.4.0
- **Déploiement** : Prêt pour Vercel / GitHub Pages

## Structure du Projet

```text
portfolio-2026/
├── assets/             # Images, photos de profil, CV (PDF), favicons...
├── index.html          # Page d'accueil : Présentation, Compétences et Parcours (Timeline)

```

## Installation & Lancement en local

Le projet étant statique (HTML/CSS/JS "vanilla"), il ne requiert aucune installation complexe ou processus de compilation (build) lourd.

1. **Cloner le dépôt** :
   ```bash
   git clone https://github.com/Mhdi-Bel/portfolio-BTS-SIO.git
   ```
2. **Ouvrir le projet** :
   - Ouvrez simplement le fichier `index.html` dans le navigateur de votre choix.
   - *Astuce (Recommandée)* : Utilisez l'extension **Live Server** sur VS Code pour bénéficier du rechargement automatique lors de vos modifications.
